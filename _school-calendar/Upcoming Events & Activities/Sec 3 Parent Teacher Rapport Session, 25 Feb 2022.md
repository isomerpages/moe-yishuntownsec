# **Sec 3 Parent Teacher Rapport Session, 25 Feb 2022**

The details for the session are as follows:

**Topic:** Sec 3 Parent Engagement Session

**Time:** Feb 25, 2022, 5.00 p.m. - 7.00 p.m.
